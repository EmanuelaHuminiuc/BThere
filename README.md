The application is intended for events, conferences and symposia. 
This application allows users to see the timeline of the event,
where everyone can post messages or pictures, there is a page for the agenda of the event,
a page for polls, a page for q & a, a page where users can see all the speakers, authors and reviewers at the event,
where they can interact with them through messages or comments.
We can also find a page with details about the location, and other details about the city where the event takes place.
For admins, various features for managing the information in the application by posting new information, viewing statistics and customizing the entire event experience.
